<h1>SigaText (Projeto Pessoal) - Macro de mensagens automáticas</h1>

SigaText é um macro de mensagens automáticas que lida diretamente com o clipboard do Windows

O script é feito em Python com o objetivo de ser uma solução indivdual da empresa para mensagens automáticas na comunicação com o cliente, economizando tempo
e facilitando a digitação.
<hr>

É uma solução individual para as mensagens de rotina da empresa, onde precisam muitas vezes enviar a mesma mensagem para contatos diferentes, 
mas com atributos diferentes, no caso a placa do veículo e o local da ocorrência
